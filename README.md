# Cursor Portfolio Setup

## Overview
This repository documents the setup process completed as part of the 100Hires recruitment portfolio challenge.

## Tools Installed
- Cursor IDE (latest version)
- GitHub

## Steps Completed
1. Installed Cursor IDE from the official website.
2. Logged into Cursor using GitHub.
3. Created a public GitHub repository.
4. Connected GitHub with Cursor.

## Challenge Encountered
The latest version of Cursor has a different interface from the instructions provided in the email. The Extensions marketplace described in the instructions is not available, so I could not install the Claude Code and Codex add-ons using the documented steps.

I contacted the recruiter to request clarification and will update this repository after receiving the correct instructions.

## Learning
This exercise helped me become familiar with GitHub repositories, Cursor IDE, and documenting technical work.
